# CodexBar Support

CodexBar Usage Monitor is maintained by HemSoft.

For support, bug reports, and feature requests, open an issue in this repository:

https://github.com/HemSoft/codexbar-ios/issues

Before opening an issue, please include:

- Your iOS or iPadOS version.
- The CodexBar app version.
- Which provider or widget is affected.
- A short description of what you expected and what happened.

Do not include API keys, access tokens, passwords, cookies, or other secrets in GitHub issues.
